# SMC Power of 3 Skill

The Power of 3 concept identifies institutional accumulation/distribution through repeated retests of key levels.

## Core Concept

Price touching a level 3 times signals:
- 1st touch: Initial reaction
- 2nd touch: Potential retest
- 3rd touch: Resolution (usually breaks out)

### Bullish Power of 3
- Support tested 3 times = accumulation
- 3rd touch often breaks above with momentum
- Target: Next resistance, measured move

### Bearish Power of 3
- Resistance tested 3 times = distribution
- 3rd touch often breaks below with momentum
- Target: Next support, measured move

## Pattern Types

### Equal Highs/Lows (Classic)
```
Touch 1: 1.0900
Touch 2: 1.0900
Touch 3: 1.0900 → Breakout
```

### Ascending/Descending 3
```
Touch 1: 1.0880
Touch 2: 1.0890
Touch 3: 1.0900 → Stronger breakout
```

### Expanding 3 (Warning)
```
Touch 1: 1.0900
Touch 2: 1.0910 (higher than 1)
Touch 3: 1.0925 (higher than 2)
= Weakening - false break likely
```

## Trading Rules

### Entry on 3rd Touch
- Enter in direction of expected break
- Stop at the 3rd touch level ± ATR/2
- Target measured move equal to the range

### Entry on Breakout
- Wait for close beyond 3rd touch level
- Enter on retest of broken level
- Stop below/above retest candle

### Measured Move
```
Range Height = Touch 3 Level - Touch 1 Level
Target = Touch 3 Level + Range Height
```

## Example

```
USDJPY H4 Power of 3:
Touch 1: 149.50 (support)
Touch 2: 149.50 (retest)
Touch 3: 149.50 (final)
↓
Breakout above 149.50
Entry: 149.55 (retest)
Stop: 149.20 (30 pips)
Range: 0 (equal touches)
Target: 149.50 + 0 = consolidation (use HT target)
Confidence: 78%
```
